\# StrayAid – Animal Emergency Operating System (AEOS)



StrayAid is a mobile-first animal rescue and welfare platform that helps people report injured, lost, or vulnerable animals quickly and connects them with nearby rescuers.



\## Tech Stack

\- React + TypeScript + Vite

\- Tailwind CSS

\- Framer Motion

\- Node.js + Express

\- PostgreSQL

\- Supabase Auth

\- Socket.io

\- Gemini Vision API
